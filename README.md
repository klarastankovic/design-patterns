# Design Patterns in C\#

A collection of clean, minimal, and well-documented C# example implementations of common software design patterns. Each example focuses on clarity and explains the core idea of the pattern with a small, runnable sample.
<br>
## Patterns included
- Creational</b>:
  - Singleton
  - Factory Method
  - Abstract Factory
  - Builder
  - Prototype

- <b>Structural</b>
  - Adapter
  - Bridge
  - Composite
  - Decorator
  - Facade
  - Flyweight
  - Proxy

- <b>Behavioral</b>
  - Chain of Responsibility
  - Iterator
  - Memento
  - Observer
  - Strategy
  - Template Method
  - Visitor
